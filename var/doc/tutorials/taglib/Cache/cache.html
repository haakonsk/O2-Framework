Caches the contents of the tag. This tag could be put around other O2-tags that require a lot of computation. If the environment variable O2CACHEROOT is not set, it will be set to O2CUSTOMERROOT/var/cache.

This will be cached 30 seconds at a time

1 + 1 = 1 + 1

Javascript/CSS will not be cached, with the exception of Javascript with "where='here'". timeout [1800] - how long (in seconds) the cache should be valid key - a unique key that is used to create the name of the cache file. content - the content to be cached Cache files are deleted/invalidated only when the files time out (according to the "timeout" attribute), so the cached content may not be up to date at all times. In other words: Use with caution!
