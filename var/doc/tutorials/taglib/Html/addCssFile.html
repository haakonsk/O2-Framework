Includes a CSS file in the head of a web page. or or file - the CSS file. where [pre] - one of {pre, here}. media - the media type, this attribute is passed on to the generated link tag. If the file name ends with ".css", this filename will be used unmodified, otherwise the file must be in the directory /css, and the extension ".css" will be appended to the given filename. <o2 header> or <o2 incStylesheet> must be called on the same page for addCssFile to work (when where="pre").
