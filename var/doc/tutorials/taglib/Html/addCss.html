Adds CSS to the style tag in the head of a web page. You can use <o2 addCss> as a normal style tag, or you can specify a class-name and the class' corresponding style in one of two ways (see the examples). When you don't use the addCss tag as a style tag, you can only specify the style for a given class-name - not for IDs or HTML tags. #item1 { color: green; } or color: red; or color: red; class - the class name param - the class name content - the style <o2 header> or <o2 incStylesheet> must be called on the same page for addCss to work.
