Adds Javascript to a script tag somewhere in a web page (depending on the value of the where attribute). var i = 14; alert(i); where [pre]. Possible values are {pre, post, onLoad, here} priority [5]. High priority javascript is included before javscript code with lower priority. Low number means high priority. content - the Javascript code If where=pre, the Javascript is included in the head of the page. If where=post, it is included near the bottom. If where=onLoad, it is executed after the page has loaded. If where=here, it is included at the place where the <o2 addJs> tag is written. If "where" is "onLoad" or "pre", <o2 header> or <o2 incJavascript> must be called on the same page in order for addJs to work. Similarly, if "where" is "post", <o2 footer> or <o2 postJavascript> must be called on the same page. See also , , , and .
