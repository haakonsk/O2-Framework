table { border-collapse : collapse; } table, td, th { border : 1px solid black; } td { padding : 2px 4px; font-family : monospace; } td + td { font-family : inherit; } With this tag, you can create slides for a tutorial or something like that. The syntax is almost exactly like that .xul-thing. But here you may use html and o2 tags, and variables, in the slides as you please, which also means that you must use the entity code (&#36;) if you want to display the dollar character.
Code Explanation
---- New page
[[PRE:something]] Preformatted text
[[EM:something]] Emphasized text
[[image src="..." width="..." height="..."]] Image
[[some url]] Link
ALIGN::[left|center|right] Alignment of current slide
HEADER::The header Header text for this slide and the following ones
FOOTER::The footer Footer text for this slide and the following ones
