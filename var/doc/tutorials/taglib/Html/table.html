Works much like a normal html table tag, but it is possible to make a table sortable. If you don't want sorting on a particular column, just set the class attribute of the table header (th) tag to "nosort". If true (1), the table headers become clickable rearrangeableRows [0] - should the table rows be rearrangeable? If true (1), you will be able to use drag and drop to rearrange the rows. Hidden input fields with name="newRowOrderForTable_<tableId>[]" will be created, one for each row in the table, and in the correct order.
