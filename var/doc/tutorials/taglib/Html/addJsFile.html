Includes a Javascript file somewhere in a web page (depending on the value of the where attribute). or where [pre]. Possible values are {pre, post, here} file - the Javascript file. If the file name starts with a slash, this filename will be used unmodified, otherwise the file must be in the directory /js, and the extension ".js" will be appended to the given filename. If where=pre, the Javascript file is included in the head of the page. If where=post, it is included near the bottom. If where=here, it is included at the place where the <o2 addJsFile> tag is written. If "where" is "pre", <o2 header> or <o2 incJavascript> must be called on the same page in order for addJsFile to work. Similarly, if "where" is "post", <o2 footer> or <o2 postJavascript> must be called on the same page. See also , , , and .
