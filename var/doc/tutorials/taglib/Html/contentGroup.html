Creates a fieldset tag with a legend (the title). Using the "disabled" attribute, an entire form inside the contentGroup tag may be disabled. When the contents of the fieldset is disabled, the fieldset's transparency is set to 40%, and if you try to click on some element in the contentGroup, an alert informs you that the contentGroup is disabled. Name: title bgColor width disabled [0] content - the content of the fieldset tag (in addition to the legend)
