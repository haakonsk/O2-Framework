Returns links to switch to other languages. Either text links or flag links. object - the object you are setting currentLocale on type [text] - one of {text, flag} orientation [horizontal] - one of {horizontal, vertical} onClick - javascript code to be performed when one of the links is clicked onSuccess - javascript code to be performed when the language has been switched class - class name of generated div tag. The div will also have a default class name of "localeSwitch". Other html attributes - ..are relayed to the div tag
