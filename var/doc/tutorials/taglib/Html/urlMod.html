Generates a URL to a page on the server. Without parameters, urlMod returns a URL to the current page. This code will generate a URL that looks something like this: /o2/System-Documentation-Taglibs/showDocumentation?module=Core&tag=call If the absoluteURL parameter is set to true (1), the URL would become something like this: http://www.my-site.com/o2/System-Documentation-Taglibs/showDocumentation?module=Core&tag=call setClass - the path to the Perl module setMethod - the method that will be called (in the Perl module) setParams - all parameters (if there should be no parameters, use removeParams) setParam - set one parameter (the others will be as in the original URL) setHash - set the hash part of the url removeParams - remove all parameters removeParam - remove the specified parameter removeHash - remove the hash part of the url appendParam - add one parameter toggleParam - toggle one parameter between a set of different values on each page urlMod is called (see example below) absoluteURL [0] The URL generated might first look like this: /o2/System-Documentation-Taglibs/showDocumentation?number=1&character=a On that page, the link will be /o2/System-Documentation-Taglibs/showDocumentation?number=2&character=b After that, number will be 3 and character will be c, then they will 1 and a, and so on.
