The config variable o2.encodeEntities determines if html special characters in variable values are translated to the corresponding entities by default or not. You can change this temporarily or permanently for the rest of the file with this tag. If encodeEntities is off, then tags inside variable values will be interpreted by the browser, Bold 'text'

$txt

This is the normal behavior: Bold 'text'

$txt

content - the text in which HTML entities in variables should not be encoded off/on/<characters to encode> - determines which (if any) of the following characters to encode in variables: <, &, >, ', ", $
Encode "on" means to encode all of the characters above except for the dollar sign.
