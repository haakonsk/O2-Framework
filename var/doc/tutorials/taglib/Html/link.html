Generates a link to. The page to link to may be set directly (through href (regular link) or src (button or image)), or the -parameters can be used. Click here or or Click here type - by default an anchor tag (<a>) is used. Type can also be set to button or image. src - The source of the image. Only used if type is image. href - What the link points to. Used if type is not specified (normal link). onClick url - Same as href, but used when type is button or image. border [0] alt - Default is the tag content (link text) confirmMsg - A message to display if the user needs to confirm the action he's about to perform activeClass - A class that will be appended to the regular class attribute if the current url equals the link's url editObject - Can be used in o2cms to link to a given object whose edit url will open in a new tab newObjectClass - Can be used in o2cms to link to a given class' new-url, which will open in a new tab startMenuItem - Can be used in o2cms to link to a url from the start menu.
