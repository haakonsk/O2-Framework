An input field based on in jquery.

Makes an input field autocompleteable. When the user types something into the autoCompleteInput, a request is sent to the backend to generate a list of values that the user can choose from.

Two input fields will be created for each autoCompleteInput, one visible and one hidden. The visible one has the same name as what you specify in the name attribute. The hidden one has the string "-hidden" appended to this name. With the attributes guiModule and method, you specify the module and method responsible for generating the list of items that the user may choose from. This method should return an array ref with the items. Each item must have a contentHtml entry. value (which will become the value of the hidden input field), onClick and class are optional. An example of this:
return [ { contentHtml => "One", value => 1 }, { contentHtml => "Two", value => 2 }, ]; name - The name of the input field. A hidden input field whose name has the string "-hidden" appended to the main input field's name will also be created. So if name="myInput", then an input field with name="myInput" and a hidden input field with name="myInput-hidden" will be created. guiModule method params onChange Attributes from jquery
