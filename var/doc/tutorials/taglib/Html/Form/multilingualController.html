MultilingualController is used in forms, to change current locale of the object you're currently editing, so that you can give different values to the same multilingual field for different languages. object - If given, the available locales are fetched from this object, otherwise they're fetched from o2.conf (o2.locales) type [text] - one of {text, flag} reloadPage [0] - Should the page be reloaded after the language has been changed? reloadConfirmMsg - The page will be reloaded only if the user says yes to this message onSwitchPre - javascript code to execute just before the language is about to change onSwitchPost - javascript code to execute just after the language has been changed If you don't like the default style, you can create a css file at O2CUSTOMERROOT/var/www/css/multilingualController.css.
