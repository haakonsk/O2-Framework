An input counter can be used to limit the number of characters the user is allowed to enter into a form field (if the maxLength attribute is set on the field). Each time a character is typed, the counter is updated. If no maximum length is provided, the counter just shows the number of characters entered. The inputCounter can be used on <o2 input> and <o2 textarea> tags. The counterId of the input field must match the id of the inputCounter. id type [span] - The kind of tag used for the counter Other html attributes - passed on to the counter tag (see the type attribute above)
