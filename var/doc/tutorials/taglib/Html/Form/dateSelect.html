A javascript calendar that allows the user to choose a date. The dateSelect tag should be defined within a form tag. When the form is submitted a variable whose name is the value of the name attribute is sent to the receiving script. The variable's value is the chosen date, in some format. <o2 header> must be defined early in the template, so the correct Javascript-files can be included, and <o2 footer /> or <o2 postJavascript /> must be included later in the template in order for the dateSelect tag to work.
