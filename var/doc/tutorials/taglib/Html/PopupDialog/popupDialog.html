Defines a popup dialog. If the linkText attribute is given, a link with that text will open the popup. If you have given the popup an ID, you can also open the popup with javascript: o2.popupDialog.display("idOfPopupDialog"); Only one popup can be displayed at one time, so to close with javascript, there's no need to specify the ID: o2.popupDialog.hide(); The content of the popup can be specified in three different ways:
  1. As content of the popupDialog tag.
  2. With popupDialog's contentId attribute, which is the ID of an HTML element.
  3. Using parameters in the popupDialog tag, or the src attribute. This causes an ajax call to fetch the popup dialog's content when it is displayed.
The implementation is based on . id - ID of the popup dialog. contentId - ID of an HTML element. If this is given, the HTML of the specified HTML element will be copied to the popup dialog and become its content (when it is displayed). submitText - the submit button's text. If submitText isn't given, there will not be a submit button. src - src of the popup dialog's content. linkText - if linkText is given a link with this text will be created. Clicking it will open the popup dialog. onClose - javascript to be executed when the close button is clicked priority [primary] - Can be primary or secondary, determines how the button is displayed text - The button text onClick - Javascript code
