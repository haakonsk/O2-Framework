Typically used to list the attributes of a tag. Style with /css/o2mlDocStyle.css. name - the name content - the content class - if class is set to "attributes", a span is put around the first word of the docListItem content (typically the attribute name), and a possible class attribute of the docListItem is set on the span tag instead of the li tag. Typical values for class is {required, optional, content, param} or a combination of those. Normal HTML unnumberd list (ul) attributes class - for attribute lists, could be one or two of {required, optional, content, param}
