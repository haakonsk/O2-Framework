The "docCode" tag puts a "code" tag around the tag content, with the "class" attribute set to whatever you specify in the "lang" attribute. If "lang" is "o2ml" or "html", the code will be highlighted. The CSS file "/css/o2mlDocStyle.css" can be used to change the highlighting style. The "docCodeResult" tag prints two links; "See generated HTML" and "See how the generated HTML is displayed on a web page". Each link can be suppressed with an attribute (see below).
