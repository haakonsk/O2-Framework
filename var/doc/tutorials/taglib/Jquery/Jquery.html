version - Jquery version. Highest version is used by default. Available versions: 1.10.2 and 1.11.1. minified {0, 1} - If the JavaScript file should be minified or not, default is not minified. important {0, 1} - If important is set to true, the given Jquery version will override previously specified Jquery versions. That does not apply if the HTML is included with AJAX, though. Default is not important.
