OFC is short for Open Flash Chart, and this is a taglib that integrates with Open Flash Chart to make very good-looking flash graphs. The API is based on <o2 gdGraph>, so by substituting gdGraph with ofcGraph and gdDataSet with dataSet, you should get a working graph. The default xLabelSeparator is a comma followed by zero or more spaces. type - [Bar] - one of {Bar, Line, 3DBar, Area, HighLowClose, Scatter, OutlinedBar, GlassBar, FadeBar, Candle, SketchBar, LineWithDots}. HighLowClose and Candle have not been implemented yet. Some of the types from GDGraph also work. title - Shown above the graph yLabel - a description of the values along the y axis yStepSize - distance between horizontal lines in the graph. A default value will be used if not given yMax - Maximum value shown on the y axis. If not given, a value near the maximum y value in the datasets is used yMin - Minimum value shown on the y axis. If not given, a value near the minimum y value in the datasets is used graphColor - default color of the graphs, may be overridden by each dataset's color attribute. If no color attributes are given, some default colors will be used bgColor - background color bgColor2 - used in combination with bgFadeAngle to have the background color change smoothly bgFadeAngle - used in combination with bgColor2 to have the background color change smoothly titleStyle [font-size: 25px] - a block of css for the title toolTip [#x_label#<br>#x_label#] - Text to show when mouse is moved over a point in the graph. You can use some special values: #val#, #key#, <br>, #x_label#, #x_legend#, #tip#. See lineWidth - default value for the width of all lines in the grid solidDots - default value (see corresponding attribute among the dataSet attributes) dotSize - default value (see corresponding attribute among the dataSet attributes) opacity - default value (see corresponding attribute among the dataSet attributes) outlineColor - default value (see corresponding attribute among the dataSet attributes) randomness - default value (see corresponding attribute among the dataSet attributes) circleSize - default value (see corresponding attribute among the dataSet attributes) content - an array reference or string containing the data for this data set. If it's a string, comma is the default separator, but this can be changed with the separator attribute description - explanatory text for this data set - will be displayed below the graph separator [/,\s*/] - If the content/dataSet isn't an array reference, but a string, then o2 needs to know where one value ends and the next one begins. The default separator is a comma followed by zero or more spaces. textSize [10] - font size of the description text (in pixels) color - color of the bars or lines in this dataset links - an array reference or a comma-separated string of links, one per value in the data set. In theory it should work with all types of graphs, but in practice, it doesn't seem to work with Scatter or Area. Line, Scatter, Area, HighLowClose, Candle, LineWithDots: width - the width of the line LineWithDots: solidDots [1] - if the dots should be solid or hollow Bar, OutlinedBar, 3DBar, GlassBar, FadeBar, SketchBar, Area, HighLowClose, Candle: opacity [80] - how non-transparent the graph is (When the mouse is moved over, the graph becomes fully opaque) Area, LineWithDots: dotSize [5] - radius of dots in pixels Scatter: circleSize - Radius of the circles (all the circles will get the same size) OutlinedBar: outlineColor [black] - color of the border of the OutlinedBar SketchBar: randomness [3] - how much the bars should deviate from perfectly drawn bars
