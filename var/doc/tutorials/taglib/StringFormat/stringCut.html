Cuts the content string after a specified number of characters. If "boundaries" is true, cuts the string at the closest word boundary before the number of characters you specified. This is equivalent to using . Hello world! content - the string to be cut param - the (maximal) number of characters to display, and optionally a string to be appended after the cut string. boundaries [0] - if true (1), the string will not be cut inside a word See also .
