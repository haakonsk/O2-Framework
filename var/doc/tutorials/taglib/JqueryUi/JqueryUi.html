version - Jquery UI version. Highest version is used by default. Available versions: 1.9.2 and 1.10.4. theme - A CSS style for the Jquery UI widgets. Available themes: black-tie, blitzer, cupertino, dark-hive, dot-luv, eggplant, excite-bike, flick, hot-sneaks, humanity, le-frog, mint-choc, overcast, pepper-grinder, redmond, smoothness, south-street, start, sunny, swanky-purse, trontastic, ui-darkness, ui-lightness and vader. Default is smoothness. minified {0, 1} - If the JavaScript file should be minified or not, default is not minified. important {0, 1} - If important is set to true, the given Jquery UI version will override previously specified Jquery UI versions. That does not apply if the HTML is included with AJAX, though. Default is not important.
