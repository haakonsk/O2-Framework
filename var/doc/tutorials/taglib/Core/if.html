Works like the equivalent construct in Perl. In addition, it can be used as the trinary operator in Perl:
1 + 1 == 2 ? "1+1 is 2" : "1+1 is not 2"
In o2ml:
<o2 if "1 + 1 == 2" then="1+1 is 2" else="1+1 is not 2" />
so, the question mark corresponds to the then attribute and the colon corresponds to the else attribute. 5 Do one thing Do a second thing instead Do something else or then - if the condition is true, this is returned else - if the condition is false, this is returned instead param - the condition
