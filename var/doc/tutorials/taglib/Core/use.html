Uses a taglib module, to make a set of tags available. The above example makes the tags defined in the module Html::Form available. This means you can use the tags <o2 input> and <o2 select>, among others. Perl looks for the module under lib/O2/Template/Taglibs, or for backend taglibs under lib/O2CMS/Template/Taglibs or for plugin taglibs under lib/O2Plugin/<pluginName>/Template/Taglibs. The core module (Core.pm) is always included and doesn't have to be 'used'.
