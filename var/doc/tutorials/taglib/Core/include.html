Includes an HTML file or an O2 template. The file path may be absolute or relative to the o2 path or the o2 customer path. To specify a relative path, "o2:" or "o2://" must be the first characters of the file path.

Parameters defined in the tag (like "pie" in the following example) are turned into variables ($pie) available for the include file to use.

file - one of three ways to specify the file name param - one of three ways to specify the file name content - one of three ways to specify the file name cacheKey - If a cache key is provided, the HTML generated by the include tag will cached. cacheTtl - How long (in seconds) the cache is valid (0 means forever)
