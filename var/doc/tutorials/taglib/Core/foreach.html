sortBy [key] - can be one of {key, value, random} for hashes, or just 'random' for arrays. In addition if an item of the array or a value of the hash is an object, it is possible to sort by a method or key in the object (see examples above). sortDirection [ascending] - must be one of {ascending, descending} sortType [string] - must be one of {string, numeric} - more to come later (date) limit - maximum number of times to go through the loop label - A name to identify the loop.
