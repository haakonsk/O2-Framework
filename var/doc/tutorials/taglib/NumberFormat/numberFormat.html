Formats a number based on the decimal format of a locale. If not specified, the default locale is used. Format, decimalSymbol and groupSymbol may also be given explicitly. 12345678.012 or 12345678.012 or 1.867 locale content - the number to be formatted format - The default depends on the locale. For Norwegian Bokmål as well as for American English, the default format is "#,##0.###" aggregateTo - If you want to add up the values from several numberFormat tags, you can specify the variable name in this parameter. Must start with a dollar sign.
