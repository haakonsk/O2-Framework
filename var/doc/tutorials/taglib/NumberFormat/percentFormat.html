Formats a number in the percent format of a locale. If not specified, the default locale is used. To get the percentage value, the input number is multiplied by 100. Format, decimalSymbol and groupSymbol may also be given explicitly. 0.95 37.5 locale isPercentAlready - If this argument is set to "1" , the input number is not multiplied by 100 content - the number to be formatted format - The default depends on the locale. For Norwegian Bokmål as well as for American English, the format it "#,##0.###" decimalSymbol - The default depends on the locale. For Norwegian Bokmål, it's ",". For American English, it's "." groupSymbol - The default depends on the locale. For Norwegian Bokmål, it's " ". For American English, it's "," See also .
