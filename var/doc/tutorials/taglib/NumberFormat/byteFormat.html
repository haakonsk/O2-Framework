Formats a number of bytes according to the decimal format of a locale. If not specified, the default locale is used. Format, decimalSymbol and groupSymbol may also be given explicitly. 12345678 locale content - the number of bytes to be formatted format - The default depends on the locale. For Norwegian Bokmål as well as for American English, the format it "#,##0.###" decimalSymbol - The default depends on the locale. For Norwegian Bokmål, it's ",". For American English, it's "." groupSymbol - The default depends on the locale. For Norwegian Bokmål, it's " ". For American English, it's "," See also .
